## 🎯 About GoFarm

GoFarm is a **complete agricultural e-commerce platform** designed to connect farmers and consumers directly. Built with Next.js 16, React 19, and TypeScript, it provides a seamless experience for buying fresh, organic farm products online.

## 🚀 Quick Start

### 📋 Prerequisites

Before you begin, ensure you have the following installed:

- **Node.js** 18.0+ ([Download](https://nodejs.org/))
- **pnpm** (recommended) or npm/yarn
- **Git** ([Download](https://git-scm.com/))

### ⚡ Installation

```bash
# 1️⃣ Extract the downloaded package
unzip gofarm-v1.0.0.zip
cd gofarm

# 2️⃣ Install dependencies
pnpm install

# 3️⃣ Set up environment variables
cp .env.example .env.local
# Edit .env.local with your credentials

# 4️⃣ Import sample data (recommended for quick start)
npx sanity dataset import seed.tar.gz production
# This imports pre-configured products, categories, brands, and sample data

# 5️⃣ Run development server
pnpm dev

# 6️⃣ Open your browser
# Visit http://localhost:3000
```

### 🔑 Required Configuration

The project requires several environment variables. Follow these steps:

1. **📖 Read the [Complete Setup Guide](./SETUP.md)** - Detailed configuration instructions
2. **🔐 Firebase Setup** - Authentication and database
3. **💳 Stripe Configuration** - Payment gateway setup
4. **📦 Sanity CMS** - Content management system
5. **📧 Email Service** - Nodemailer with Gmail OAuth2
